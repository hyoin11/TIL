### 지식
19일 화상면접때 HTTP 메소드에 관해 설명하라고 하였다.
알고있던 GET, POST, PUT, DELETE를 말하였는데 PATCH에 대해서 질문하였다. PATCH를 들어보긴 했지만 보통 GET, POST를 사용하였어서 질문에 대답을 못하였다.
PUT은 데이터가 있으면 덮어쓰고, 없으면 새로 만들어줌. PATCH는 데이터의 일부분을 수정

또한 REST에 대한 개념을 다시 정리하였다.
URI를 통해 자원을 명시하고, HTTP 메서드를 통해 해당 자원의 상태를 주고받는 것

WEB WAS에 대한 차이
WEB은 정적 컨텐츠를 제공. WAS는 동적 컨텐츠를 제공

모든 개념에 대해서 명확하게 알아가자.
