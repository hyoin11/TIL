### spring core basic
SRP(Single Responsibility Principle) - 단일 책임의 원칙
하나의 클래스는 하나의 책임만 가져야 함

DIP(Dependency Inversion Principle) - 의존관계 역전 원칙
추상화에 의존. 구체화에 의존해서 안됨. 의존성 주입은 이 원칙을 따르는 방법 중 하나

OCP(Open Closed Principle) - 개방 폐쇠 원칙
확장에는 열려있으나 변경에는 닫혀있어야 함

컨테이너
AppConfig 처럼 객체를 생성하고 관리하면서 의존관계를 연결해주는 것
